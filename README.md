# 通讯录管理系统
基于SQL Server的通讯录管理系统，C#实现，中间注册增加了一个通过html+Servlet实现的网页注册(可以局域网注册)

登陆面板,会有一个默认的头像，用户注册的时候可以选择头像，也可以在菜单里更换头像，在登录面板再次登录时会更新头像
